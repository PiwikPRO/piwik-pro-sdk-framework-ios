## Reporting a Vulnerability

If you discover a vulnerability in this repository, please use the **“Report a vulnerability”** feature available under the **Security** tab of the repository.  
- Go to the repository on GitHub.  
- Click on the **Security** tab.  
- Select **Report a vulnerability** to securely notify us.  

This ensures that sensitive information is not publicly disclosed before a fix is available.  

## Response Timeline
- You will receive an acknowledgment of your report within **48 hours**.
